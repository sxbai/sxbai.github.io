利用replit搭建Alist

发表于2022-11-11|更新于2022-11-13|course
|阅读量:

前言：

replit 前面介绍过了，在这里就不赘述了，官网地址：https://replit.com

Alist 官方介绍 A file list program that supports multiple storage, powered by Gin and Solidjs. / 一个支持多存储的文件列表程序，使用 Gin 和 Solidjs，挂载网盘等等的程序，开源项目地址：https://github.com/alist-org/alist

废话不多说，开始第一步

【1】创建容器

【2】输入命令行

然后在这里输入下面的命令然后Enter 回车，去GitHub获取最新的版本搭建，这里（2022年11月10日）以当前最新的3.3.0为例演示

wget https://github.com/alist-org/alist/releases/download/v3.3.0/alist-linux-amd64.tar.gz

然后在这里输入下面的命令然后Enter 回车

tar -zxvf alist-linux-amd64.tar.gz

然后在这里输入下面的命令然后Enter 回车

chmod +x alist

然后在这里输入下面的命令然后Enter 回车

./alist server

最后把./alist server 添加到main.sh Ctrl+S保存起来。

最后点击Run启动Alist项目就行了。

【3】打开Alist 的项目网址

这样Alist就是搭建好了，剩下的就是美化，挂载网盘了，就不提了。

官方的挂载网盘文档可以看看 https://alist.nn.ci/zh/guide/drivers

强烈建议搭配上一篇文章来使用 uptime来监控repl项目防止休眠 https://sxbai.com/2022/11/11/mogenius-uptime-kuma
